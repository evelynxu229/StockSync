# StockSync
**A State-of-the-Art Microservice Project Based on Spring Boot 3**

StockSync is an advanced microservice project, grounded in Spring Boot 3 and harnessing a suite of cutting-edge technologies. It seamlessly integrates the robust capabilities of Spring Cloud 3.0+ with a range of pivotal technologies such as Eureka for service discovery, Keycloak for secure authentication, Circuit Breaker for system resilience, Kafka for efficient message processing, Docker for containerization, and both PostgreSQL and MongoDB for versatile data storage. This fusion of technologies culminates in a comprehensive and secure backend website.

## How to Run
1. Run mvn clean package -DskipTests to build the applications and create the docker image locally.
2. Run docker-compose up -d to start the applications.
